   
    # introprg
    Repositorio de ejercicios de Introducción a la Programación (con el sistema de aprendizaje y corrección de ejercicios PRGTEST) desarrollados durante el  CFGS de Desarrollo de Aplicaciones Multiplataforma (DAM).
    
    
    Lenguaje principal: Java  
    Autor: Kilian Fernández
=============================
    
    ## Sobre este repositorio

    Este repositorio recoge mi progreso real en programación desde los primeros ejercicios hasta los bloques más avanzados del curso.  

    Está organizado por carpetas numeradas que reflejan la evolución del nivel y la dificultad de los problemas.

    Es un repostiroio de práctica intensiva y constante sobre:
    - lógica algorítmica
    - manipulación de strings y arrays
    - recursividad
    - modularización
    - análisis de caracteres
    - resolución de errores (debugging)

    El objetivo principal ha sido aprender a pensar los problemas antes de programarlos, no solo hacer que el código funcione.
