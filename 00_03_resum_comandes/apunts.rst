#################################
Exercici 00_03. Resum de comandes
#################################

Autoria: Kilian Fernández – 23/09/2025

Introducció: Aquest exercici proposa emplenar la taula amb les primeres comandes de Git i Linux apreses a clase.


    Comanda	                                                       Descripció
---------------                 --------------------------------------------------------------------------------
$ mkdir carpeta/	            crea una carpeta anomenada carpeta/
$ cd carpeta/	                canvia el directori actual a carpeta/
$ cat fitxer	                mostra el contingut del fitxer per pantalla
$ gedit fitxer &	            obre el fitxer amb l’editor de text gedit en segon pla (no bloqueja la terminal)
$ ls	                        llista els fitxers i carpetes del directori actual
$ rm                            fitxer	elimina el fitxer anomenat fitxer
$ git init	                    permet inicialitzar un repositori git
$ git clone	                    descarrega una còpia d’un repositori remot (GitHub, etc.) al teu ordinador
$ git status	                mostra l’estat actual del repositori (canvis, fitxers nous, etc.)
$ git add	                    afegeix fitxers a l’índex per preparar-los per al commit
$ git commit	                crea un snapshot (punt de control) dels canvis amb un missatge descriptiu
$ git push	                    envia els commits locals al repositori remot (p. ex. GitHub)
$ git pull	                    descarrega i fusiona els canvis del repositori remot amb el local
